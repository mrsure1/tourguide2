## 📋 API 신청 방법

### Step 1: 공공데이터포털 회원가입

1. **사이트 접속**: https://www.data.go.kr
2. **회원가입 클릭** (우측 상단)
3. **개인회원 선택**
4. **본인인증** (휴대폰 또는 아이핀)
5. **회원정보 입력** 후 가입 완료

---

### Step 2: API 검색

**방법 1: 기관명으로 검색**
```
검색창에 입력:
- 기업마당
- 중소벤처기업부
- 창업진흥원
- 보조금통합포털
```

**방법 2: 키워드로 검색**
```
검색창에 입력:
- 중소기업 지원사업
- 정책자금
- 창업 지원
- 보조금
```

---

### Step 3: 활용신청

1. **API 상세 페이지**에서 "활용신청" 버튼 클릭
2. **신청 정보 입력**
   - 활용목적: `정책자금 매칭 웹 서비스 개발`
   - 상세기능: `사용자 프로필 기반 맞춤형 정책자금 추천 서비스`
3. **신청 버튼** 클릭
4. **승인 대기** (보통 즉시~2시간)

---

### Step 4: API 키 확인

1. **마이페이지** 접속
2. **오픈API → 개발계정** 메뉴
3. 승인된 API의 **일반 인증키 (Encoding)** 복사

## 🚀 프로젝트 적용 방법

### 1. 환경 변수 설정

프로젝트 루트에 `.env.local` 파일 생성:

```env
# 기업마당 API
NEXT_PUBLIC_BIZINFO_API_KEY=여기에_API_키_입력

# K-Startup API
NEXT_PUBLIC_KSTARTUP_API_KEY=여기에_API_키_입력

# 보조금통합포털 API
NEXT_PUBLIC_SUBSIDY_API_KEY=여기에_API_키_입력
```

---

### 2. 개발 서버 재시작

```bash
# 터미널에서 Ctrl + C로 서버 중지
npm run dev
```

---

### 3. API 클라이언트 구현

현재 프로젝트에 이미 API 클라이언트 구조가 구축되어 있습니다:
- 파일 위치: `lib/api/publicDataApi.ts`
- API 키 설정 후 자동으로 실제 데이터 사용
